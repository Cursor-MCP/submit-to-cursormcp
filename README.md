# 🎉 Submit To cursormcp.net 🚀

欢迎您为 [你的导航站名] 提交新的网站资源！

## 如何提交？

1.  点击上方的 [Issues] 标签页。
2.  点击绿色的 [New issue] 按钮。
3.  选择适合的模板（例如：**「提交新网站」**），并填写所需信息。
4.  点击 [Submit new issue] 即可。我们会在审核后尽快处理！

## 提交规范

为了帮助我们高效处理，请确保您的提交包含以下信息：
- **网站名称：**
- **网址（URL）：**
- **分类：** （例如：工具、设计、学习...）
- **简介：** 一句话描述这个网站是做什么的。

感谢您的贡献！
